El programa que resuelve el problem de Adoquinamiento tiene una implementación gráfica con tkinter para python3
Requerimientos:
    Python3
        Bibliotecas externas de Python:
            numpy
            tkinter
        Bibliotecas usualmente incluidas en python default:
            sys
            random
            math

Instalación:
    Para instalar pip basta:
    $sudo apt pip install Python3-pip
    Y para las librerias:
    $pip install <libreria>
Finalmente para correr la práctica:
    $python3 src/P1.py
